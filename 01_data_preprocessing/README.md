# Исследование надёжности заёмщиков

## Описание проекта
Цель проекта — исследовать, какие факторы влияют на вероятность невозврата кредита, и подготовить данные для дальнейшего анализа.  
Проект ориентирован на **предобработку данных и исследовательский анализ (EDA)** без применения моделей машинного обучения.

Задача типична для кредитного скоринга и аналитики в финансовых организациях: на основе клиентских данных выявить зависимости между социально-демографическими признаками и платёжной дисциплиной.

## Используемые данные
Датасет содержит информацию о заёмщиках:
- семейное положение  
- количество детей  
- уровень дохода  
- цель кредита  
- наличие/отсутствие задолженности  

(Датасет предоставлен в рамках учебного кейса.)

## Основные этапы работы
1. **Первичный анализ данных**
   - просмотр структуры датасета
   - анализ типов данных и распределений  
2. **Предобработка данных**
   - обработка пропусков
   - исправление аномалий и ошибок
   - приведение данных к корректным типам
   - устранение неявных дубликатов  
3. **Категоризация и feature engineering**
   - группировка доходов
   - классификация целей кредита
4. **Аналитические срезы**
   - анализ зависимости задолженности от:
     - семейного положения
     - количества детей
     - уровня дохода
     - цели кредита
5. **Выводы**
   - формулировка факторов, повышающих и понижающих риск задолженности
