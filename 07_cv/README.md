# 07_cv — Computer Vision

## Определение возраста покупателей по фото (Age Estimation)

**Задача:** по фотографии лица оценить возраст человека (задача регрессии).  
**Цель:** построить модель компьютерного зрения, которая предсказывает возраст с приемлемой ошибкой.

---

### Данные
- Датасет с фотографиями людей и разметкой возраста (`labels.csv`)
- Размер выборки: **7 591** изображение  
- Разбиение: **train 4 113 / valid 1 500 / test 1 978**

> Сами изображения в репозиторий не загружаю (ограничения курса/датасета). В ноутбуке остаются инструкции по воспроизведению.

---

### Подход
- EDA: распределение возраста, примеры изображений, базовые выводы по датасету
- Предобработка + аугментации (train):
  - `RandomHorizontalFlip`, `RandomRotation`, `ColorJitter`
  - нормализация/препроцессинг из предобученных весов torchvision
- Transfer learning на предобученных CNN (torchvision)
- Сравнение нескольких архитектур

---

### Модели
Проверены:
- **ResNet50** (pretrained)
- **EfficientNet-B0** (pretrained)

---

### Метрика и результат
**Метрика:** MAE (Mean Absolute Error), в годах.

Лучший результат:
- **EfficientNet-B0 — Test MAE ≈ 7.9**

(Для сравнения в ноутбуке также есть прогоны других конфигураций/архитектур.)
