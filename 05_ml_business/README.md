# 05_ml_business — Machine Learning for Business

Проекты по ML с выраженным **бизнес-контекстом**: прибыль/риски/принятие решений на основе предсказаний моделей.  
Фокус — не только качество модели, но и **интерпретация результата с точки зрения бизнеса**.

---

## Проекты

### 1) Выбор региона для бурения скважин: оценка прибыли и рисков (Bootstrap)
**Файл:** `Выбор региона для бурения скважин, оценка прибыли и рисков (Bootstrap).ipynb`

**Задача:** выбрать регион для бурения, который даст максимальную ожидаемую прибыль при допустимом уровне риска.

**Бизнес-контекст:**
- ограниченный бюджет на разработку скважин
- необходимость учитывать риск убытков
- решение принимается на основе прогнозов и вероятностных оценок

**Что сделано:**
- подготовка и анализ данных геологоразведки
- обучение модели для предсказания объёма запасов сырья
- расчёт прибыли по выбранным скважинам
- оценка рисков и доверительных интервалов методом Bootstrap
- сравнение регионов по ожидаемой прибыли и риску убытков
- обоснование финального бизнес-решения

**Модель:** линейная регрессия  
**Метрики/методы:** RMSE, Bootstrap, доверительные интервалы, вероятность убытков

---

### 2) Прогноз истощения абсорбента NaOH в абсорбере (промышленная аналитика)
**Файл:** `Прогноз истощения абсорбента NaOH в абсорбере (промышленная аналитика).ipynb`

**Задача:** оценить скорость истощения NaOH и **прогнозировать остаточный ресурс абсорбера** до порогового значения концентрации (планирование замен/обслуживания).

**Бизнес-контекст:**
- снижение простоев и аварийных остановок
- планирование обслуживания по фактическому состоянию
- прогнозирование времени до “пороговой” концентрации реагента

**Что сделано:**
- очистка и фильтрация данных по операциям оборудования и пробам
- расчёт наработки между пробами (суммарная длительность операций)
- расчёт изменения концентрации NaOH между измерениями
- построение простой модели зависимости “падение NaOH ↔ наработка”
- расчёт прогноза оставшегося времени/объёма до порогового уровня

**Модель:** линейная регрессия  
**Метрики/методы:** R² (качество аппроксимации), оценка остаточного ресурса по порогу
