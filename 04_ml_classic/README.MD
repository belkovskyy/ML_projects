# 04_ml_classic — Classic Machine Learning (Tabular)

Проекты по классическому ML на табличных данных: **подготовка данных → признаки → обучение → валидация → метрики → интерпретация**.  
Здесь есть и **классификация**, и **регрессия**.

---

## Проекты

### 1) Прогнозирование оттока клиентов банка (Classification)
**Файл:** `Прогнозирование оттока клиентов банка на основе машинного обучения.ipynb`  
**Цель:** предсказать факт ухода клиента (удержание/маркетинг).  
**Ключевое:** дисбаланс классов, сравнение моделей, подбор параметров.  
**Модели:** Logistic Regression, Decision Tree, Random Forest  
**Метрики:** F1 (основная), ROC-AUC (доп.)

---

### 2) Рекомендательная система тарифов мобильной связи (Classification)
**Файл:** `Рекомендательная система тарифов мобильной связи на основе машинного обучения.ipynb`  
**Цель:** рекомендовать тариф по признакам поведения пользователя.  
**Ключевое:** train/valid/test, сравнение моделей, подбор гиперпараметров.  
**Модели:** Logistic Regression, Decision Tree, Random Forest  
**Метрика:** Accuracy

---

### 3) Оценка цен на недвижимость (Regression) — 2 проекта
Два проекта на схожую задачу (оценка стоимости), но на **разных данных/контекстах**.

#### 3.1) Оценка цен на недвижимость (СПБ)
**Файл:** `Оценка цен на недвижимость (СПБ).ipynb`  
**Цель:** предсказать цену/стоимость объекта недвижимости.  
**Ключевое:** предобработка, feature engineering, сравнение бустингов, интерпретация.  
**Модели:** Linear Regression, Ridge, CatBoost, LightGBM  
**Метрики:** RMSE / MAE  
**Интерпретация:** SHAP

#### 3.2) House Prices (Kaggle) — оценка стоимости недвижимости
**Файл:** `Оценка цен на недвижимость (Kaggle House Prices).ipynb`  
**Цель:** предсказать `SalePrice` (Kaggle House Prices).  
**Ключевое:** обработка пропусков/категорий, лог-трансформация таргета, сравнение моделей.  
**Модели:** Linear Regression, Ridge, CatBoost, LightGBM, XGBoost  
**Метрики:** RMSE / MAE  
**Интерпретация:** SHAP
